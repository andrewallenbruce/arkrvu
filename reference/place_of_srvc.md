# Place of Service

Place of Service

## Usage

``` r
place_of_srvc
```

## Format

### `place_of_srvc`

A data frame with 99 rows and 4 columns:

- pos_code:

  Place of Service Code

- pos_type:

  Place of Service Type

- pos_name:

  Place of Service Name

- pos_description:

  Place of Service Description
