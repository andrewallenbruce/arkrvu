# Mount [pins](https://pins.rstudio.com/reference/pins-package.html) board

Mount [pins](https://pins.rstudio.com/reference/pins-package.html) board

## Usage

``` r
mount_board(source = c("local", "remote"))
```

## Arguments

- source:

  `<chr>` `"local"` or `"remote"`

## Value

`<pins_board_folder>` or `<pins_board_url>`
