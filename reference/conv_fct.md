# Conversion Factor

Conversion Factor

## Usage

``` r
conv_fct
```

## Format

### `conv_fct`

A data frame with 36 rows and 6 columns:

- date_start:

  Effective Date

- date_end:

  End Date

- date_iv:

  Date Interval

- cf:

  Conversion Factor

- chg_abs:

  Absolute Change

- chg_rel:

  Relative Change
