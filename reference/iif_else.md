# ifelse wrapper using kit::iif

ifelse wrapper using kit::iif

## Usage

``` r
iif_else(x, yes, no)
```

## Arguments

- x:

  `<lgl>` vector

- yes, no:

  Values to return depending on TRUE/FALSE element of `x`. Must be same
  type and be either length 1 or same length of `x`.

## Value

`<chr>` vector
